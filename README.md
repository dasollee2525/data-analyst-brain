# DA Brain

현직 Data Analyst의 온톨로지 기반 지식 두뇌.

어떤 회사에 가든 꺼내 쓸 수 있는, "상황 → 판단 → 행동" 사고 루틴 체계.

## 노트 타입

| 타입 | 폴더 | 설명 |
|------|------|------|
| MOC | `00_MOC/` | 지식 축별 진입점 & 인덱스 |
| Concept | `01_Concept/` | 개념 하나당 노트 하나 |
| Situation | `02_Situation/` | 실무 반복 문제 패턴 |
| Case | `03_Case/` | 실제 경험 기록 |
| Inbox | `04_Inbox/` | 빠른 캡처 (미분류) |

## 지식 축

1. **분석 방법론** — 인과추론, 베이지안, A/B 테스트
2. **지표 설계 & 트래킹** — North Star, 이상 탐지
3. **예측 & 불확실성** — 모델 선택, 피처 설계

## 링크 3원칙

1. 새 노트 → MOC에 추가
2. Case → Concept + Situation 역링크
3. Concept에 "실제 케이스" 섹션 필수 (최소 링크 2개)
