## 한 줄 정의

베이지안 구조적 시계열 모델로 "처치가 없었다면 지표가 어떻게 됐을까"를 합성해, 마케팅/기능 변화의 지표 임팩트를 사후 측정하는 방법. (Google, 2015)

## 언제 쓰는가 (전제 조건)

- A/B 테스트 없이 이미 전체 배포된 변화의 효과를 측정해야 할 때
- 처치 시점이 명확한 단일 이벤트 (캠페인 시작일, 기능 출시일 등)
- 처치 집단과 유사하게 움직이는 **공변량(covariates)**이 있을 때
  - 예: 처치 없는 다른 지역/채널/제품 지표
- 처치 전 충분한 시계열 데이터 확보

## 작동 원리

1. **처치 전 기간**: 처치 집단 지표 + 공변량으로 베이지안 구조적 시계열 모델 학습
2. **처치 후 기간**: 학습된 모델로 "처치가 없었다면" 예측 (counterfactual)
3. **효과 = 실제값 - 예측 counterfactual**
4. 베이지안 posterior로 효과의 불확실성 구간 자동 제공

```
          처치 전 학습          처치 후 예측
지표 ─────────────────|──────────────────
                      ↑               ← 실제값
                   처치 시점          ← counterfactual (예측)
                                      효과 = 둘의 차이
```

## DiD와의 차이

| | DiD | CausalImpact |
|---|---|---|
| 데이터 형태 | 패널 (다수 집단×시점) | 시계열 (단일 집단+공변량) |
| 통제군 구성 | 명시적 통제 집단 | 공변량으로 합성 |
| 불확실성 | 표준오차 | 베이지안 posterior |
| 계절성 처리 | 수동 | 자동 (구조적 모델) |

## 핵심 함정

1. **공변량 선택이 결과를 좌우**: 처치에 영향받지 않는 공변량을 써야 함 — 처치와 함께 오염된 변수를 쓰면 counterfactual이 틀림
2. **처치 시점 이외의 동시 이벤트**: 처치와 동시에 다른 변화가 있었다면 효과 분리 불가
3. **처치 전 fit 확인 필수**: 처치 전 기간에 예측이 실제값을 잘 따라가지 못하면 처치 후 예측도 신뢰 불가
4. **단기 효과만 측정**: 처치 후 기간이 짧으면 장기 효과는 추정 불가
