## 한 줄 정의

처치(treatment)와 통제(control) 집단을 무작위 배정해 인과 효과를 추정하는 실험. 무작위화가 선택 편향을 제거한다.

## 언제 쓰는가 (전제 조건)

- 무작위 배정이 가능한 상황 (온라인 제품, 가격 테스트 등)
- 샘플 크기가 충분히 확보 가능할 때
- 실험 효과가 비교적 단기간에 측정 가능할 때

**사전 체크리스트**:
- [ ] 검정력 분석으로 필요 샘플 크기 계산
- [ ] MDE(최소 탐지 가능 효과) 사전 정의
- [ ] 1차 지표 사전 고정 (HARKing 방지)
- [ ] 실험 단위(user vs session vs device) 결정

## 핵심 함정

1. **SUTVA 위반**: 처치 집단이 통제 집단에 영향 (네트워크 효과, 가격 테스트)
2. **Peeking 문제**: 중간에 결과를 보고 조기 중단 → 1종 오류 폭증
3. **SRM (Sample Ratio Mismatch)**: 실제 배정 비율이 설계와 다른 경우 → 랜덤화 버그 의심
4. **Novelty 효과**: 새로운 것에 반응하는 단기 효과 → 충분한 실험 기간 필요
5. **다중 비교**: 지표를 여러 개 볼수록 우연히 유의한 결과가 나올 확률 증가
